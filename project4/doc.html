Project 4

Documentation

For this project I used my knowledge of HTML, CSS, and Javascript as well as some online resources to help me with format and code. I used stackoverflow.com to find information on the javascript for image movement using arrow keys and image movement with a button click. For the CSS animation in Example 2, I used w3schools.com for help. I also used normalize.css and skeleton.css for some parts of the site. For the tardis svg image, I based the code off an example on codepen.io. Using codepen and w3schools I learned a lot about how to make new svg shapes and how to resize and position rectangles and polygons in order to make the shape of the TARDIS.
